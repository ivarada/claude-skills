# Citation Format Guide - Academic Publications

This guide explains the citation formats generated by the Crossref lookup skill.

## APA 7th Edition Format

### Journal Article Format

**Basic Structure:**
```
Author, A. A., Author, B. B., & Author, C. C. (Year). Title of article. Title of Journal, volume(issue), pages. https://doi.org/xx.xxxx/xxxxx
```

**Examples:**

**Single author:**
```
Hellsten, U. (2013). The genome of the Western clawed frog Xenopus tropicalis. Nature, 496(7446), 311-316. https://doi.org/10.1038/nature12373
```

**Two authors:**
```
Smith, J., & Jones, M. (2020). Machine learning applications in healthcare. Journal of Medical Informatics, 45(3), 234-245. https://doi.org/10.1234/example
```

**Three or more authors:**
```
Brown, A., Davis, B., Wilson, C., & Taylor, D. (2019). Climate change impacts on biodiversity. Environmental Science, 78(2), 145-167. https://doi.org/10.1234/example
```

**21+ authors:**
```
Smith, J., Brown, A., Wilson, C., Davis, M., Lee, K., Chen, L., ... Johnson, R. (2021). Large-scale genome study. Nature Genetics, 53(4), 401-420. https://doi.org/10.1234/example
```

### Book Format

**Basic Structure:**
```
Author, A. A. (Year). Title of book. Publisher. https://doi.org/xx.xxxx/xxxxx
```

**Example:**
```
Murphy, K. P. (2012). Machine learning: A probabilistic perspective. MIT Press. https://doi.org/10.1234/example
```

### Book Chapter Format

**Basic Structure:**
```
Author, A. A. (Year). Title of chapter. In Title of Book (pp. xx-xx). Publisher. https://doi.org/xx.xxxx/xxxxx
```

**Example:**
```
Zhang, Y. (2018). Deep learning architectures. In Handbook of Artificial Intelligence (pp. 45-78). Springer. https://doi.org/10.1234/example
```

### Conference Paper Format

**Basic Structure:**
```
Author, A. A. (Year). Title of paper. Conference Name. Publisher. https://doi.org/xx.xxxx/xxxxx
```

**Example:**
```
Bahdanau, D., Cho, K., & Bengio, Y. (2015). Neural machine translation by jointly learning to align and translate. ICLR 2015. https://doi.org/10.1234/example
```

### Key APA 7 Rules for Academic Works

1. **Author names**: Last name, First initial. Middle initial.
2. **Multiple authors**: 
   - 2 authors: Use &
   - 3-20 authors: Comma-separated, & before last
   - 21+ authors: First 19, ..., last author
3. **Title**: Sentence case (only first word and proper nouns capitalized)
4. **Journal names**: Title case, italicized
5. **Volume**: Italicized
6. **Issue**: In parentheses, not italicized
7. **DOI**: Always include as URL (https://doi.org/...)
8. **No retrieval date needed** for DOI links

## BibTeX Format

### Journal Article (@article)

```bibtex
@article{hellsten_2013_genome,
  author = {Hellsten, Uffe and Smith, Jane and Doe, John},
  title = {The Genome of the Western Clawed Frog Xenopus Tropicalis},
  journal = {Nature},
  year = {2013},
  volume = {496},
  number = {7446},
  pages = {311-316},
  doi = {10.1038/nature12373}
}
```

### Book (@book)

```bibtex
@book{murphy_2012_machine,
  author = {Murphy, Kevin P.},
  title = {Machine Learning: A Probabilistic Perspective},
  publisher = {MIT Press},
  year = {2012},
  isbn = {978-0262018029},
  doi = {10.1234/example}
}
```

### Book Chapter (@inbook)

```bibtex
@inbook{zhang_2018_deep,
  author = {Zhang, Yao},
  title = {Deep Learning Architectures},
  booktitle = {Handbook of Artificial Intelligence},
  publisher = {Springer},
  year = {2018},
  pages = {45-78},
  doi = {10.1234/example}
}
```

### Conference Paper (@inproceedings)

```bibtex
@inproceedings{bahdanau_2015_neural,
  author = {Bahdanau, Dzmitry and Cho, Kyunghyun and Bengio, Yoshua},
  title = {Neural Machine Translation by Jointly Learning to Align and Translate},
  booktitle = {ICLR 2015},
  year = {2015},
  doi = {10.1234/example}
}
```

### Other Types (@misc, @techreport, @phdthesis)

```bibtex
@misc{smith_2020_preprint,
  author = {Smith, John},
  title = {Novel Approaches to Climate Modeling},
  year = {2020},
  publisher = {arXiv},
  doi = {10.48550/arXiv.2020.12345}
}

@techreport{jones_2019_technical,
  author = {Jones, Mary},
  title = {Technical Report on System Performance},
  institution = {NASA},
  year = {2019},
  number = {NASA-TR-2019-001}
}

@phdthesis{wilson_2021_dissertation,
  author = {Wilson, Robert},
  title = {Advanced Topics in Quantum Computing},
  school = {Stanford University},
  year = {2021}
}
```

### Citation Key Format

The scripts generate citation keys as: `firstauthor_year_firstword`

**Examples:**
- `hellsten_2013_genome`
- `murphy_2012_machine`
- `bahdanau_2015_neural`

### BibTeX Field Reference

**Required for @article:**
- author, title, journal, year

**Required for @book:**
- author, title, publisher, year

**Required for @inproceedings:**
- author, title, booktitle, year

**Optional but recommended:**
- doi, volume, number, pages, isbn, url

## Special Cases

### No Date Available

**APA7:**
```
Author, A. A. (n.d.). Title of work. Publisher. https://doi.org/xx.xxxx
```

**BibTeX:**
```bibtex
@article{author_nd_title,
  author = {Author Name},
  title = {Title of Work},
  year = {n.d.}
}
```

### Online-Only Publication

**APA7:**
```
Author, A. A. (2020). Title of article. Journal Name, 5(3), Article e12345. https://doi.org/xx.xxxx
```

### Preprint/Arxiv

**APA7:**
```
Author, A. A. (2021). Title of preprint. arXiv. https://doi.org/10.48550/arXiv.xxxx.xxxxx
```

**BibTeX:**
```bibtex
@misc{author_2021_title,
  author = {Author Name},
  title = {Title of Preprint},
  publisher = {arXiv},
  year = {2021},
  doi = {10.48550/arXiv.xxxx.xxxxx}
}
```

### Dataset

**APA7:**
```
Author, A. A. (2020). Title of dataset (Version 1.0) [Data set]. Repository Name. https://doi.org/xx.xxxx
```

**BibTeX:**
```bibtex
@misc{author_2020_dataset,
  author = {Author Name},
  title = {Title of Dataset},
  howpublished = {Data set},
  year = {2020},
  doi = {10.xxxx/xxxxx}
}
```

## Using Citations

### For APA7 Citations in Word/Google Docs

1. Generate citation using the script
2. Copy the formatted citation
3. Paste into your reference list
4. Ensure alphabetical order
5. Use hanging indent (0.5 inches)
6. Double-space if required

### For BibTeX in LaTeX

1. Create a `references.bib` file
2. Add all BibTeX entries
3. In your `.tex` file:
   ```latex
   \documentclass{article}
   \usepackage{natbib}
   
   \begin{document}
   
   As shown by \citet{hellsten_2013_genome}...
   Recent work \citep{murphy_2012_machine} demonstrates...
   
   \bibliographystyle{apalike}
   \bibliography{references}
   \end{document}
   ```
4. Compile with: `pdflatex`, `bibtex`, `pdflatex`, `pdflatex`

### For Zotero/Mendeley

1. Generate BibTeX entries
2. Save to `.bib` file
3. Import: File → Import → Select file
4. Use Zotero's citation plugin in Word/Google Docs
5. Convert to any citation style you need

## Verification Checklist

Before submitting work with citations:

- ✅ All DOIs are correct and clickable
- ✅ Author names are properly formatted
- ✅ Publication years are accurate
- ✅ Journal/conference names are complete
- ✅ Volume/issue numbers match the original
- ✅ Page numbers are included (if available)
- ✅ Special characters are properly escaped (BibTeX)
- ✅ Citations are in alphabetical order (APA)
- ✅ Hanging indent is applied (APA)

## Common Issues

**Missing abstract:**
- Not all Crossref records include abstracts
- Check the publisher's website

**Incomplete author information:**
- Some records only have last names
- Verify authors from the original publication

**Multiple publication dates:**
- Online vs print dates may differ
- Use the date most relevant to your citation context
- Crossref provides: published, published-print, published-online

**Preprints vs published versions:**
- Always check if a preprint was later published
- Cite the final published version when available

**Conference paper variations:**
- Some conferences have proceedings published as journals
- Follow the format of how it was actually published
